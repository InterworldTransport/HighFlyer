Overview Information
Project Description HighFlyer is a java class bundle with several small working applications designed to simulate and support lighter than air vehicle operation in the upper atmosphere.
Application Architecture and Planning Documentation
 All Project files are in the CVS repository.

Browse through them or check out a working copy if you know how to use CVS. We will use mailing lists and an IRC channel [#highflyer] on irc.openprojects.net to discuss ideas for this project if the number of developers and users becomes large enough to make the effort worth it. Until then, you may send direct email to the project administrator(s).

Published documentation can be found in the Doc Manager when it is ready for the users. within the project. Documentation that does not face the user directly may be found within the repository mostly under the docs directory.

Once a group of documents has reached a moderately evolved state, they will be released using the same file release feature a project such as ours uses to release the actual software and source code.
